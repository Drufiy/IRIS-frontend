# IRIS Frontend

Marketing website and frontend foundation for IRIS.

## Scope

This repo currently contains:

- the M2 website implementation
- the frontend design foundation docs
- the shared visual language for the future desktop app

The implementation is aligned to the current backend repo at:

- `https://github.com/Drufiy/IRIS-backend`

## Stack

- Vite
- HTML
- CSS
- vanilla JavaScript

## Local Development

```powershell
npm install
npm run dev
```

## Build

```powershell
npm run build
```

## Structure

- `index.html` - marketing site shell
- `src/main.js` - state animation and reveal logic
- `src/styles.css` - shared visual system and responsive layout
- `docs/frontend/` - M0 and M1 design docs
