Changelog

How the frontend is evolving into a full IRIS surface.

This site did not start as a complete launch layer. It has been moving through deliberate milestones so the visual language, product story, and desktop direction stay aligned.

M2-M3 Core aesthetic, brand language, and product polish
M4-M5 Multi-page site and stronger product framing
M6-M7 Desktop direction and command-center story
M8 Launch-ready surrounding pages and release surfaces

Milestone History

The frontend path so far.

M2

Website foundation

Established the initial public-facing look, the quiet sci-fi tone, and the landing-page identity for IRIS.

M3

Experience polish

Improved visual rhythm, motion, and overall product confidence so the site felt intentional instead of placeholder.

M4

Multi-page expansion

Added dedicated product pages so capabilities, safety, and later desktop direction could each breathe.

M5

Product framing upgrade

Moved the website closer to a real product surface with stronger narrative, stack framing, and desktop setup for later milestones.

M6-M7

Desktop narrative and command center story

Introduced the desktop direction page and began showing the future app as an operating surface, not just a future chat window.

M8

Launch-ready website layer

Added download, getting started, FAQ, changelog, and waitlist surfaces so the website can now function as a fuller launch shell before desktop app implementation.

IRIS

Voice-first AI operating layer for real desktop workflows.
